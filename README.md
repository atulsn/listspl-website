## 📌 About This Project
This is a fully redesigned concept version of the official **LIST Software Pvt. Ltd.** website — a Core Banking Software company based in Sangli, Maharashtra, India, serving 100+ co-operative banks since 1997.

The goal of this project was to modernize the look and feel of the original website while keeping all the original content, information, and branding intact.

---

## ✨ What Was Improved / Added

| Feature | Original Site | This Redesign |
|---|---|---|
| Design | Outdated | Modern, clean UI |
| Mobile Responsive | Basic | Fully responsive |
| Navigation | Broken links | All links working |
| Solutions Section | Basic cards | Expandable detail panels |
| Contact Section | Single form | 3 branch locations + form |
| Request a Demo | Not available | Modal with branch details |
| Career Section | Basic form | Redesigned with perks info |
| Awards Section | Basic images | Lightbox image viewer |
| Clients Section | Logo grid | Interactive hover cards |
| Footer | Placeholder text ("abc") | Clean, complete footer |
| Copyright Year | Outdated (2009) | Updated (2025) |

A fully redesigned, modern website for **LIST Software Pvt. Ltd.**, a Core Banking Software company based in Sangli, Maharashtra, India.

## 🚀 Live Preview
Host this on GitHub Pages, Netlify, or Vercel — just upload and go!

```
listspl-website/
├── index.html        ← Main website (single file, fully self-contained)
└── README.md         ← This file
```

## 🌐 How to Host on GitHub Pages
1. Create a new GitHub repository (e.g. `listspl-website`)
2. Upload `index.html` and `README.md` to the repository
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save** — your site will be live at:
   `https://yourusername.github.io/listspl-website/`

## 🎨 Tech Stack
- Pure **HTML5 + CSS3 + Vanilla JavaScript**
- No frameworks, no dependencies
- Google Fonts (DM Sans + Playfair Display) — loaded via CDN
- Fully **mobile responsive**
- Works in all modern browsers
